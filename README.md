# HTML-PAINT

Простой холст на чистом JS. Рисуйте мышкой, выбирайте цвет и толщину линии с возможностью отмены действий.

## Демо
https://amainn.github.io/html-paint/

## Возможности
- Рисование мышью на canvas
- Выбор цвета
- Регулировка толщины линии
- Очистка холста
- Отмена/возврат действий

## Технологии
- HTML5 Canvas
- CSS3
- JavaScript

## Запуск локально
1. Скачайте ZIP архив репозитория
2. Откройте `index.html` в браузере
